# HB9CV 天线 (HB9CV Phased Array)

## 1. 概述
HB9CV 是一款经典的 **2单元相位阵列天线**，由瑞士无线电爱好者 Rudolf Baumgartner (HB9CV) 在 1950 年代设计。
* **核心特点**：**全驱动 (All Driven)**。与八木天线（只有一根有源振子）不同，HB9CV 的两个振子都通过相位线连接到馈源。
* **优势**：在仅有 2 个单元的情况下，提供了超越普通 3 单元八木的增益和极高的前后比。

## 2. 结构与相位原理
HB9CV 的两个振子间距通常为 **$\lambda/8$** (即 45° 电长度)。
为了实现单向辐射，电流必须满足特定的相位关系：
* **前振子 (Front)**：电流滞后。
* **后振子 (Rear)**：电流超前。

通常通过 **Z 形交叉相位线** 来实现供电，使得两个振子上的电流相位差约为 **$225^\circ$**。
* **相位构成**：$180^\circ$ (交叉线反相) + $45^\circ$ (相位线长度) = $225^\circ$。

## 3. 电场叠加公式 (Field Summation)
假设两个振子上的电流幅度相等 ($I_1 \approx I_2 = I$)，远场辐射总电场 $E_{total}$ 可以表示为两个振子产生的电场矢量之和。

$$E_{total}(\theta) = E_{element}(\theta) \times AF(\theta)$$

其中 $AF(\theta)$ 是**阵列因子 (Array Factor)**：

$$AF(\theta) = 1 + e^{j(kd \cos \theta + \delta)}$$

* $k = \frac{2\pi}{\lambda}$ (波数)
* $d = \frac{\lambda}{8}$ (间距) $\Rightarrow kd = \frac{\pi}{4} = 45^\circ$
* $\delta$ (电流相位差) $\approx -225^\circ$ (前振子滞后后振子)
* $\theta$：方位角 ($0^\circ$ 为前方，$180^\circ$ 为后方)

## 4. 方向图分析 (Radiation Pattern)

我们将参数代入阵列因子公式来验证其前后比：

### 4.1 前向辐射 ($\theta = 0^\circ$)
$$kd \cos(0^\circ) = 45^\circ$$
总相位角 $\psi = 45^\circ + (-225^\circ) = -180^\circ$。
$$AF(0^\circ) = 1 + e^{j(-180^\circ)} = 1 - 1 = 0 \quad \text{(Wait?)}$$

*(注：实际工程中 HB9CV 的相位配置是让后向抵消。上述推导假设了简单的直接相加。实际上 HB9CV 的相位差 $\delta$ 设置为使得 $\theta=180$ 时相位对消。让我们反推一下最佳相位)*

**修正后的相位条件**：
为了让后方 ($180^\circ$) 抵消：
空间相位滞后 $kd \cos(180) = -45^\circ$。
总相位 $\psi_{back} = -45^\circ + \delta$。
若要抵消，需 $\psi_{back} = 180^\circ$。
$\Rightarrow \delta = 225^\circ$ (后振子超前前振子 225度，或前振子超前后振子 135度)。

**正确的工作模式**：
* **后方 ($180^\circ$)**：前振子的信号传播到后方（滞后45度），与后振子信号（本来就滞后/超前特定角度）正好反相抵消。$E_{back} \approx 0$。
* **前方 ($0^\circ$)**：后振子的信号传播到前方（滞后45度），与前振子信号同相叠加（或接近同相）。

其归一化方向图幅度公式近似为：
$$F(\theta) \approx \cos\left( \frac{\pi}{8} (1 - \cos \theta) \right) \quad \text{(简化心形模型)}$$

这个公式产生了一个非常锐利的心形，屁股后面（180度）有一个极深的零点。

## 5. HB9CV vs 八木
| 特性 | HB9CV (2单元) | 普通八木 (2单元) |
| :--- | :--- | :--- |
| **驱动方式** | 双振子有源 | 单振子有源，一根寄生 |
| **增益** | 高 (~6.5 - 7 dBi) | 中 (~5 - 6 dBi) |
| **前后比** | **极高 (20-40 dB)** | 一般 (10-15 dB) |
| **带宽** | 较宽 | 较窄 |
| **制作难度** | 难 (需制作相位线) | 易 |

**总结**：HB9CV 是“小而强”的典范，虽然结构复杂（由于相位线），但其优异的指向性和抗干扰能力使其成为 6米和 2米波段测向和通联的神器。
