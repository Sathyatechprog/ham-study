# 磁环天线 (Magnetic Loop / Small Loop)

## 1. 概述
磁环天线（小环天线）是一种**电尺寸极小**的天线（周长 $C < \lambda/10$）。
* **结构**：通常是一个垂直架设的单匝或多匝线圈，顶部或底部串联一个可变电容进行调谐。
* **特点**：效率低、带宽极窄（高 Q 值）、但在抗干扰接收方面表现卓越。

## 2. 物理模型：磁偶极子 (Magnetic Dipole)
小环天线在物理上等效于一个**磁偶极子**。
* **电流分布**：由于尺寸远小于波长，我们可以假设环上的电流 $I$ 是**均匀分布**的（这一点与偶极子天线两端电流为0完全不同）。
* **辐射机制**：它主要辐射磁场分量（近场），在远场转换为电磁波。

## 3. 远场电场公式 (Electric Field Formula)
对于一个小环（半径为 $a$，面积 $A=\pi a^2$），其在球坐标系下的远场电场分量 $E_\phi$ 为：

$$E_\phi = \frac{\eta k^2 I A}{4\pi r} e^{-jkr} \sin\theta$$

其中：
* $\eta \approx 120\pi \approx 377\Omega$ (自由空间波阻抗)
* $k = 2\pi/\lambda$ (波数)
* $I$ 是环上的电流
* $A$ 是环的面积
* $r$ 是距离
* $\theta$ 是相对于**环轴线 (Z轴)** 的夹角

## 4. 辐射方向图 (Radiation Pattern)
从电场公式中可以看出，方向性函数为：

$$F(\theta) = \sin\theta$$

这里的 $\theta$ 定义非常关键：
* **$\theta$ 是相对于环面法线（轴线）的夹角**。

这意味着：
* **$\theta = 0^\circ$ (轴线方向/垂直于环面)**：$\sin(0)=0$。辐射为 **0 (零点 Null)**。
    * *通俗解释*：就像通过望远镜筒看东西，如果你从侧面看环（看到一条线），信号最强；如果你正对着环的圆心看（看到一个圆），信号最弱。
* **$\theta = 90^\circ$ (环面方向/侧面)**：$\sin(90)=1$。辐射最大。

**形状描述**：
* **立体图**：像一个甜甜圈（Donut），但是甜甜圈是套在轴线上的。
* **俯视图 (Azimuth)**：呈现完美的**“8”字形**。最大辐射方向与环面平行，零点方向与环面垂直。

## 5. 核心优势：深零点抗干扰
磁环天线最值钱的地方不在于它能发多远（发射效率通常很低），而在于它的接收能力。
由于其方向图具有极其锋利的零点（Nulls），我们可以通过旋转天线，将干扰源（如邻居家吵闹的开关电源）置于天线的轴线方向（零点）。
此时，干扰信号会被大幅削弱（可达 -20dB 到 -30dB），从而把微弱的有用信号从噪音中“捞”出来。
