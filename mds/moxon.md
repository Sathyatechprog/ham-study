# Moxon 矩形天线 (Moxon Rectangle Antenna)

### 1. 概述
Moxon 矩形天线是一种两单元的**线束定向天线**。
* **特点**：尺寸紧凑（约为同频八木天线的 70%），具有惊人的**前后比 (Front-to-Back Ratio)**。
* **应用**：手持测向（猎狐）、抗干扰通信、卫星通信、受限空间安装。

### 2. 结构组成
它由两个 U 形弯折的金属线组成，互相对置，形成一个矩形：
* **驱动单元 (Driven Element)**：矩形的“前”半部分，通过馈线供电。
* **反射单元 (Reflector Element)**：矩形的“后”半部分，无源寄生，比驱动单元略长。
* **关键间隙 (Gap)**：两个单元末端之间的微小缝隙。

### 3. 工作原理：电容耦合的魔法
Moxon 的核心在于那个**缝隙 (Gap)**。
* **区别于八木**：传统的两单元八木天线，反射器和驱动器离得较远，主要依靠互感耦合。
* **Moxon 机制**：将振子末端向内弯折并彼此靠近，引入了显著的**电容耦合**。这种耦合改变了电流在反射器上的分布和相位。
* **结果**：这种特定的相位关系使得后向辐射被剧烈抵消（相消干涉），从而形成了极深的后向零点。

### 4. 辐射方向图公式 (Mathematical Model)
虽然精确的 Moxon 辐射场需要复杂的数值分析（如 NEC 建模），但其标志性的**心形方向图 (Cardioid Pattern)** 可以用以下简化函数近似描述：

$$F(\theta) \approx \left( \frac{1 + \cos \theta}{2} \right)^A$$

其中：
* $\theta$ 是方位角（$0^\circ$ 为天线前方，即驱动单元方向）。
* $A$ 是形状因子（通常 $A \approx 1.3$）。
* **前向辐射 ($\theta = 0^\circ$)**：$\cos(0)=1 \Rightarrow F(0)=1$ (最大值)。
* **后向零点 ($\theta = 180^\circ$)**：$\cos(180)=-1 \Rightarrow F(180)=0$ (完全抵消)。

这个公式从数学上解释了为什么它的“屁股”后面听不到声音。

### 5. 核心优势
1.  **天然 50Ω 匹配**：这是 Moxon 最神奇的地方。通过调整缝隙宽度，输入阻抗可以直接调到 **50Ω**。**不需要**任何伽马匹配、发夹匹配或巴伦，直接接同轴电缆即可获得极低的驻波。
2.  **完美的“盾牌”**：它的后瓣极小（心形方向图），前后比通常能达到 **20dB - 30dB**。这意味着它能像盾牌一样屏蔽身后的干扰信号。
3.  **宽波束**：前方的主瓣很宽（约 130°），不需要像激光一样精确瞄准，大概方向对了就能通联。

### 6. Moxon vs 八木 (2单元)
| 特性 | Moxon 矩形天线 | 2单元八木天线 |
| :--- | :--- | :--- |
| **尺寸** | 小 (紧凑矩形) | 大 (长条形) |
| **前后比** | **极高 (>20dB)** | 一般 (10-15dB) |
| **增益** | 中等 (~5.5 dBi) | 略高 (~6.0 dBi) |
| **匹配** | **直连 50Ω** | 通常需要匹配网络 |
| **带宽** | 极宽 | 较窄 |

**一句话总结**：Moxon 天线是“低调的实力派”，用极其简单的结构实现了直连 50Ω 和超高前后比，是 DIY 入门和便携通联的极佳选择。
